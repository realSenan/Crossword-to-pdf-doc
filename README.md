Crossword PDF Exporter

This project allows users to create a crossword puzzle and export it as a high-quality PDF.

Project Overview

Word Placement:
Words entered by the user are automatically placed on a grid in horizontal and vertical directions.
Intersecting letters are automatically aligned correctly.
Starting cells are numbered automatically.

Clues and Solution:
Lists of horizontal and vertical clues are generated automatically.
Solution letters can be hidden or shown as needed.

PDF Export:
The grid and clues are exported in SVG format into the PDF, ensuring crisp and sharp output.
The grid scales automatically to fit A4 pages.
Multi-page PDF support is included.

Error Handling:
Users are alerted if the title is missing or if the crossword has not been generated.
